# Noah Haeger – Portfolio Website

## Overview
This project is a personal portfolio website designed to showcase my experience, projects, and technical skills in finance and business analytics. It serves as a centralized, professional space where recruiters and employers can quickly understand my background and the type of work I have done. The site emphasizes clean design, accessibility, and clarity, reflecting how I approach data-driven decision-making. It is intended for professionals in finance, consulting, and analytics roles who want a concise view of my capabilities.

## Features
- Responsive, mobile-first design that works across devices  
- Clean, professional layout with strong visual hierarchy  
- Project-focused sections highlighting tools, work, and impact  
- Accessible structure following basic WCAG guidelines  
- Simple navigation with anchor links for easy browsing  

## Future Improvements
- Add additional projects with deeper analytics and financial modeling  
- Introduce interactive elements using JavaScript  
- Expand content with case-study style project breakdowns  
- Improve visual polish and animations for a more premium feel  
- Integrate a contact form for direct outreach
